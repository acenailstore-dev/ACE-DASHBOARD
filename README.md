# ACE Nails Supply — Lightspeed Dashboard

Dashboard kết nối trực tiếp Lightspeed Retail cho ACE Nails Supply.

## Deploy lên Vercel (5 phút)

### Bước 1: Upload lên GitHub
1. Vào github.com → New repository → Đặt tên `ace-dashboard` → Create
2. Upload toàn bộ folder này lên repo đó

### Bước 2: Deploy Vercel
1. Vào vercel.com → Add New Project
2. Chọn repo `ace-dashboard` vừa tạo
3. Click Deploy → Done

### Bước 3: Xong
Vercel sẽ cho anh 1 URL dạng `ace-dashboard.vercel.app` — truy cập là thấy dashboard.

## Tính năng
- 📊 Overview: doanh thu hôm nay, tháng này, tổng sản phẩm, khách hàng
- 💰 Sales: tất cả đơn hàng tháng này
- 📦 Products: danh sách sản phẩm + SKU + giá
- 👥 Customers: khách hàng mới nhất
- 🗂 Inventory: tồn kho theo sản phẩm
